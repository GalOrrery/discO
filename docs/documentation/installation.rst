============
Installation
============

|Code Size|

**********
With `pip`
**********

.. container::

	|PyPI| |PyPI Format|

The easiest way to get *disc_expand* is to install with `pip <https://pypi.org/project/disc_expand/>`_. To install with pip::

    pip install disc_expand


***********
From source
***********

To install the latest version, or from a branch, *disc_expand* can be installed from `source <https://github.com/nstarman/disc_expand.git>`_.

From the command line.
::

	cd folder/in/which/to/install

	git clone https://github.com/cwru-pat/disc_expand.git

	cd disc_expand

	python setup.py install
